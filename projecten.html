Readable Reader

De readable reader is een web applicatie voor mensen met slecht zicht. Met deze applicatie zouden ze makkelijker boeken moeten kunnen lezen. Deze applicatie leest de epub bestanden en zorgt ervoor dat deze aangepast worden naar de wensen van de lezer.

Voor dit project is React + Redux gebruikt voor de frontend, de backend is gedaan met nodejs. De database die we hebben gebruikt is mongoDB.

Veiling Website

Voor dit project is de frontend in html gedaan met het bootstrap framework. De backend is gedaan in php. De database die hier gebruikt werd was mssql.

Helaas is de website offline en kan het eind resultaat niet geshowd worden. De foto's komen groot en deels overeen met het orgineel.
